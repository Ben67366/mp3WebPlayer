# 波形オーディオプレイヤー

単一のHTMLファイルだけで動作する、高機能な波形表示オーディオプレイヤーです。

MP3 / WAVファイルを読み込むと波形を美しく可視化し、波形をクリックしたりズームしたりしながら任意の位置から再生できます。また、**SRT / VTT / JSON形式の字幕**を読み込んで、再生位置に同期してハイライト表示・クリックでジャンプすることができます。

![スクリーンショットが必要な場合は自分で撮影して追加してください]

## ✨ 主な機能

- **美しい波形表示** — Canvasでリアルタイム描画。ダークモダンなデザイン
- **自由なズーム & スクロール** — 波形の横幅を自由に拡大・縮小（320px〜9600px）
- **直感的なシーク** — 波形上をクリック/ドラッグで再生位置を変更
- **字幕同期表示** — SRT/VTT/JSON対応。現在の再生位置の字幕を自動ハイライト
- **字幕リストからジャンプ** — リストをクリックするとその時間に移動
- **話者ラベル付け（PC専用）** — 「ラベル付け」ボタンで会話の話者 A/B/C… を行ごとに付与。人数に応じてチェック／ラジオ／ドロップダウンが切り替わる。SRT には `[A] テキスト` 形式で保存
- **再生速度変更** — 0.75x / 1x / 1.5x などプリセット + 自由選択
- **キーボードショートカット充実**
- **ボリュームスライダー**
- **完全オフライン動作** — 外部依存なし（TailwindはCDNのみ使用）
- **字幕トランスクリプション連携** — 特に [whisper_transcribe](https://github.com/Ben67366/whisper_transcribe) で生成した字幕と相性が良い
- **モバイル対応** — スマホ画面でコンパクトに表示（タイトル・余白・ボタン・波形を縮小）。字幕リストは独立スクロール可能。
- **文字サイズ調整** — 字幕エリアに − / + ボタンでフォントサイズを自由に変更（10px〜28px）
- **サンプルボタン** — 「サンプルを試す」ボタンで ./sample/sample.mp3 を自動ロード + ./sample/sample.srt があれば字幕も自動で読み込み（デモ用）

## 🚀 使い方

### 1. プレイヤーを開く
- `waveform-player.html` をブラウザで開くだけです（ローカルファイルで動作します）

### 2. 音声ファイルを読み込む
- アップロードゾーンに **MP3 / WAV** ファイルをドラッグ＆ドロップ
- または「ファイルを選択」ボタンで選択

### 3. 字幕を読み込む（任意）
1. プレイヤー下部の **「字幕ファイル読込」** ボタンをクリック
2. 以下の形式のファイルを指定：
   - `.srt`
   - `.vtt`
   - `.json`（`[{start, end, text}, ...]` の配列形式）

字幕が読み込まれると右側（または下部）にリストが表示され、再生に同期して現在の字幕が強調表示されます。

### 4. 話者ラベル付け（PC・任意）

1. 画面幅が広いとき（タブレット横〜PC）に表示される **「ラベル付け」** を押す
2. 参加者数を **− / ＋** で調整（名前は固定で **A, B, C…**。自由入力なし。初期は1人）
3. 各字幕行で話者を選択
   - **1人** … チェックボックス
   - **2〜3人** … ラジオボタン（「なし」あり）
   - **4人以上** … ドロップダウン
4. **「SRTを保存」** でラベルを反映した `.srt` を書き出し
   - Chrome / Edge などでは、読込時に取得したファイルへ**上書き**できる場合があります
   - 非対応環境では保存ダイアログまたはダウンロードになります

ラベル形式の例:

```
1
00:00:00,910 --> 00:00:01,910
[A] ももたろう。
```

> スマホ幅ではラベル付け UI は表示されません（操作が難しいため）。

## ⌨️ キーボードショートカット

| キー          | 動作                     |
|---------------|--------------------------|
| `Space`       | 再生 / 一時停止          |
| `←` / `→`     | 5秒戻る / 5秒進む        |
| `1`           | 再生速度 1.0x            |
| `2`           | 再生速度 1.5x            |
| `0`           | 再生速度 0.75x           |

波形表示中はマウスホイールやドラッグで横スクロールも可能です。

## 📄 対応字幕フォーマット

- **SRT**（最も一般的）
- **VTT**（WebVTT）
- **JSON**（シンプルな配列形式）

```json
[
  { "start": 0.0, "end": 3.2, "text": "こんにちは" },
  { "start": 3.5, "end": 7.8, "text": "今日は良い天気ですね" }
]
```

> `whisper_transcribe` ツールで生成した `.srt` / `.vtt` はそのまま利用できます。

## 🛠 技術スタック

- 純粋な **Vanilla JavaScript**（依存なし）
- **HTML5 `<audio>`** + **Web Audio API**（波形解析）
- **Canvas** による波形描画
- **Tailwind CSS**（CDN経由）
- すべての処理がブラウザ内で完結

## 📦 デプロイ / 配布方法

このプレイヤーは**ビルド不要**の単一HTMLファイルです。

### おすすめの使い方
- GitHub Pages でそのまま公開
- ローカルネットワークで共有（`python -m http.server` など）
- ブラウザの「ファイルを開く」で直接利用

> 注意: 一部のブラウザではローカルファイルからのオーディオ再生に制限がある場合があります。その場合はローカルサーバー経由で開いてください。

## 🔗 関連プロジェクト

- **[whisper_transcribe](https://github.com/Ben67366/whisper_transcribe)** — Faster-Whisper large-v3-turbo を使った高精度字幕生成CLIツール（SRT / VTT / TXT出力）

この波形プレイヤーと組み合わせることで、音声 → 字幕生成 → 視覚的確認・編集 のワークフローが非常に快適になります。

## 📝 カスタマイズ

- 波形の色や高さはHTML内のCSS変数やJavaScriptの `drawWaveform` 関数で調整可能です。
- 字幕の表示形式を変えたい場合は `renderSubtitles` 周辺のDOM操作を編集してください。
- より高度な編集機能が必要な場合は別途拡張してください。
